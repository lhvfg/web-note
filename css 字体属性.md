### css 字体属性

- 字体系列

  font-family: '字体'

- 字体大小

  font-size: xxpx

- 加粗

  font-weight: number(不加单位，400为正常,700为加粗)

- 文字样式

  font-style:

- 复合属性

  font: font-style font-weight font-size/line-height font-family

  - 每个位置对应的属性固定 直接写属性值
  - 用空格而不是分号隔开，最后加一个分号
  - 除了文字大小和字体(最后两个)，其他都可省略





### 注意

1. 大范围和小范围都指定了格式，小范围以精准指定为准
